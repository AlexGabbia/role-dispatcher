# Cybersecurity / Sicurezza Informatica

| # | Role | Description | Reports To |
|---|------|-------------|------------|
| 1 | CISO (Chief Information Security Officer) | Executive responsible for corporate cybersecurity strategy | CEO / CTO |
| 2 | Security Architect | Designs the overall security architecture of an organization | CISO |
| 3 | Security Engineer | Implements and manages security tools and solutions | Security Architect / CISO |
| 4 | Cybersecurity Analyst | Monitors and analyzes threats to IT system security | Security Engineer / CISO |
| 5 | SOC Analyst | Works in the Security Operations Center, monitoring events in real time | Security Engineer / CISO |
| 6 | Application Security Engineer | Ensures code and application security during development | Security Architect / Engineering Manager |
| 7 | Identity & Access Management Specialist | Manages authentication and authorization systems | Security Engineer / CISO |
| 8 | Incident Response Analyst | Manages and coordinates response to cybersecurity incidents | Security Engineer / CISO |
| 9 | Penetration Tester | Simulates cyberattacks to identify system vulnerabilities | Security Engineer / CISO |
| 10 | Compliance Analyst | Verifies that systems comply with security and privacy regulations (GDPR, ISO 27001) | CISO / IT Director |
| 11 | Threat Intelligence Analyst | Collects and analyzes information on emerging cyber threats | Security Engineer / CISO |
| 12 | Red Team Operator | Conducts advanced simulated attacks to test organizational defenses | Security Architect / CISO |
| 13 | Blue Team Analyst | Actively defends systems and improves detection capabilities | Security Engineer / CISO |
| 14 | Forensic Analyst | Investigates cyber incidents, collecting digital evidence | Security Engineer / CISO |
| 15 | Cryptographer | Designs and analyzes cryptographic algorithms to protect communications | Security Architect / CISO |
