# IT Support & Administration / Supporto e Amministrazione IT

| # | Role | Description | Reports To |
|---|------|-------------|------------|
| 1 | IT Service Manager | Manages IT service delivery following frameworks like ITIL | IT Director / CIO |
| 2 | IT Coordinator | Coordinates daily IT activities and liaises with other departments | IT Service Manager / IT Director |
| 3 | IT Support Specialist | Resolves hardware and software technical issues for enterprise users | IT Service Manager / IT Coordinator |
| 4 | IT Auditor | Verifies IT system compliance with regulations and corporate policies | IT Director / CISO |
| 5 | IT Compliance Officer | Ensures the IT organization complies with applicable laws and regulations | IT Director / CISO |
| 6 | IT Asset Manager | Tracks and manages the inventory of all corporate IT resources | IT Service Manager / IT Director |
| 7 | Help Desk Technician | Provides first-level technical support to users | IT Support Specialist / IT Service Manager |
| 8 | IT Procurement Specialist | Manages purchases of hardware, software and IT services | IT Director / IT Asset Manager |
| 9 | NOC Technician | Monitors networks and systems from the Network Operations Center | IT Operations Manager / Network Engineer |
| 10 | Desktop Support Technician | Configures and maintains workstations (PCs, printers, peripherals) | IT Support Specialist / IT Service Manager |
| 11 | License Manager | Manages software licenses, ensuring compliance and cost optimization | IT Asset Manager / IT Director |
| 12 | Field Technician | Provides technical assistance directly at the customer's site | IT Service Manager / IT Support Specialist |
| 13 | IT Secretary / IT Assistant | Supports the IT department with administrative and organizational tasks | IT Director / IT Coordinator |
