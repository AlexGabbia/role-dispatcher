# Specialized Roles / Altre Figure Specializzate

| # | Role | Description | Reports To |
|---|------|-------------|------------|
| 1 | Digital Transformation Manager | Leads the digital transformation process of an organization | CIO / CEO |
| 2 | Data Protection Officer (DPO) | Ensures organizational compliance with data protection regulations (GDPR) | CEO / CIO |
| 3 | Technical Recruiter | Specialized in recruiting technical and IT profiles | VP of Engineering / HR Director |
| 4 | DevRel (Developer Relations) | Builds relationships with the developer community, creating content and support | CTO / CMO Digital |
| 5 | IT Vendor Manager | Manages relationships with IT service and product vendors | IT Director / CIO |
| 6 | IT Legal Advisor | Provides legal counsel on technology topics: software contracts, intellectual property | CIO / IT Director |
| 7 | Accessibility Engineer | Implements technical solutions to make software accessible to everyone | Engineering Manager / Product Designer |
| 8 | Site Builder / Webmaster | Builds and maintains websites, often using CMS like WordPress | IT Director / Content Manager |
| 9 | Localization Engineer | Adapts software and content for different markets and languages | Engineering Manager / Product Manager |
| 10 | Open Source Maintainer | Manages and coordinates the development of open source software projects | CTO / Engineering Manager |
| 11 | Scrum Product Coach | Helps organizations improve product management in the Agile context | VP of Engineering / Agile Coach |
| 12 | Quantitative Analyst (Quant) | Uses mathematical and computational models for financial analysis and algorithmic trading | CDO / CTO |
| 13 | Bioinformatics Engineer | Applies computer science to biology for genomic and biomedical data analysis | Engineering Manager / CTO |
| 14 | AI Ethics Specialist | Evaluates the ethical impact of artificial intelligence systems | CDO / CTO |
| 15 | IT Sustainability Manager | Manages environmental sustainability of IT infrastructure | CIO / IT Director |
| 16 | No-Code Automation Specialist | Automates business processes with no-code tools (Zapier, Make) | IT Director / IT Operations Manager |
| 17 | Digital Accessibility Auditor | Verifies website compliance with accessibility standards (WCAG) | Accessibility Specialist / Product Designer |
| 18 | GIS Specialist | Develops and manages geographic information systems for spatial data analysis | IT Director / Engineering Manager |
