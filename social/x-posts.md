### Thread 2/5 - The Problem
```
The problem: AI coding tools give you ONE generalist perspective.

But real projects need specialists — a Security Engineer thinks differently than a Frontend Developer.

Role Dispatcher automatically identifies which experts your request needs and dispatches them as agents.
```

### Thread 3/5 - How It Works
```
How it works:

1. You make a request (in any language)
2. It matches against 185+ roles across 13 categories
3. Proposes 1-3 specialist agents
4. You confirm or adjust
5. Agents collaborate with handoff notes
6. A review agent validates the output

All automatic. Zero config.
```
